# PallasbilED

Static website for Pallas Bilvård.

## Files

- `index.html` — site homepage
- `priser.html` — price list page
- `screen.png` — optional image asset

## Deployment

This is a static website and can be deployed directly on Vercel.

### Deploy on Vercel

1. Sign in to Vercel and link the repository `albaniansniper/pallasbilED`.
2. Choose the `main` branch.
3. Set the root directory to the repository root.
4. No build command is required for static HTML.
5. Set the output directory to `/`.

The default Vercel static deployment will serve `index.html` automatically.
